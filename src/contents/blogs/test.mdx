## Penjelasan

Tutorial ini menunjukkan cara membuat button interaktif dengan HTML, CSS, dan JavaScript. Ketika button diklik, warnanya akan berubah dan teks akan menampilkan jumlah klik.

## Langkah-langkah Pembuatan

### 1. Struktur HTML

Pertama, kita buat struktur HTML dasar dengan sebuah button:

```html
<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Button Interaktif</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <div class="container">
        <button id="color-button" class="btn">
            <span>Klik Saya</span>
        </button>
        <p class="hint">Klik button untuk melihat perubahan warna</p>
    </div>
    
    <script src="script.js"></script>
</body>
</html>
```

### 3. Interaktivitas dengan JavaScript

Terakhir, kita tambahkan JavaScript untuk membuat button interaktif dan berubah warna saat diklik:

```javascript
// Mendapatkan referensi ke button
const button = document.getElementById('color-button');
const buttonText = button.querySelector('span');
let clickCount = 0;

// Array warna yang akan digunakan secara bergantian
const colors = ['red', 'green', 'purple', 'yellow', ''];

// Listener untuk event click pada button
button.addEventListener('click', function() {
    // Menambahkan counter
    clickCount++;
    buttonText.textContent = \`Klik ke-\${clickCount}\`;
    
    // Mendapatkan class warna saat ini
    let currentColorClass = '';
    for (const color of colors) {
        if (color && button.classList.contains(color)) {
            currentColorClass = color;
            break;
        }
    }
    
    // Mendapatkan index warna berikutnya
    const currentIndex = colors.indexOf(currentColorClass);
    const nextIndex = (currentIndex + 1) % colors.length;
    
    // Menghapus class warna saat ini
    if (currentColorClass) {
        button.classList.remove(currentColorClass);
    }
    
    // Menambahkan class warna berikutnya
    if (colors[nextIndex]) {
        button.classList.add(colors[nextIndex]);
    }
    
    // Membuat efek riak air (ripple)
    createRippleEffect(event);
});

// Fungsi untuk membuat efek riak air (ripple)
function createRippleEffect(event) {
    const x = event.clientX - button.getBoundingClientRect().left;
    const y = event.clientY - button.getBoundingClientRect().top;
    
    const ripple = document.createElement('span');
    ripple.classList.add('ripple');
    ripple.style.left = \`\${x}px\`;
    ripple.style.top = \`\${y}px\`;
    
    button.appendChild(ripple);
    
    // Menghapus elemen ripple setelah animasi selesai
    setTimeout(() => {
        ripple.remove();
    }, 500);
}
```
